# ZAPHs_MiniMod
ZAPHs_MiniMod for 7 Days to Die V2.6

# Summary 0.0.1

This MOD adds the brand NEW bratwurst item and recipe to the game! Some other recipes for existing items / resources are added as well!

Details

This MOD adds the brand NEW bratwurst item and recipe to the game!
- NEW bratwurst item (campfire)
- NEW bratwurst recipe item
  - both can be found at the traders
  - both can be found in loot

Some other recipes for existing items / resources are added as well!
- resourceCoal recipes (campfire, chemistryStation)
- toolBeaker recipe (forge)
- resourceSpring recipe (forge)
- smallEngine recipe (workbench)
- resourceAcid recipe (chemistryStation)

Localization
- English
- German

Unzip ZAPHs_MiniMod.zip into a folder\
Copy the ZAPHs_MiniMod folder into your Mods directory
